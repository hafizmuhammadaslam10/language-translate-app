# üåç Google Translate API App (Node.js + HTML + CSS)

A simple, clean translation tool using **Node.js**, **Express**, and the unofficial but **free and API-keyless** [`google-translate-api-x`](https://www.npmjs.com/package/google-translate-api-x) library. Users can enter text, choose a destination language (or enter a custom one), and see the translated result side-by-side ‚Äî all in a beautiful responsive UI.
